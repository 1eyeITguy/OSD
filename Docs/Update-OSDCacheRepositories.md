# Update-OSDCacheRepositories

## SYNOPSIS
Updates the OSDCache repository in $env:ProgramData\OSDCache from the GitHub Origin.

## SYNTAX

```
Update-OSDCacheRepositories [-Force] [<CommonParameters>]
```

## DESCRIPTION
Updates the OSDCache repository in $env:ProgramData\OSDCache from the GitHub Origin.
