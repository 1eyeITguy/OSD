# Get-DellWinPEDriverPack

## SYNOPSIS
Returns the URL of the latest Dell WinPE 11 Driver Pack

## SYNTAX

```
Get-DellWinPEDriverPack [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Returns the URL of the latest Dell WinPE 11 Driver Pack

## EXAMPLES

### EXAMPLE 1
```
Get-DellWinPEDriverPack
```

### EXAMPLE 2
```
$DellWinPEDriverPack = Get-DellWinPEDriverPack
```
