# Add-WinPEDrivers

## SYNOPSIS

## SYNTAX

```
Add-WinPEDrivers [<CommonParameters>]
```

## DESCRIPTION
Adds the WinPEDrivers repository in the OSDCache at $env:ProgramData\OSDCache from the GitHub Origin.
